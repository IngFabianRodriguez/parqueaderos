# RF-157 — Configuración de Notificaciones y Plantillas

## Metadata

- **Módulo**: 13 — Admin Panel
- **Prioridad**: Media
- **Tipo**: Funcional
- **Servicios afectados**: notification-service, admin-service

---

## User Story

**Como** admin de tenant **quiero** configurar las notificaciones que se envían a clientes y operadores **para** mantener una comunicación efectiva y personalizada.

---

## Objetivo

El panel permite activar/desactivar tipos de notificación (email, SMS, push), personalizar plantillas de mensajes, y configurar reglas de envío (timing, déclenche). Cada tenant puede adaptar las comunicaciones a su marca.

---

## Actores

- **Actor principal**: Admin de tenant
- **Actor secundario**: Supervisor de sede

---

## Flujo Funcional

### Activar/Desactivar Tipo de Notificación

1. Admin accede a "Configuración > Notificaciones".
2. Ve lista de eventos notificables con toggle de activo/inactivo.
3. Admin activa los deseados y guarda.
4. Sistema aplica cambios inmediatamente.

### Editar Plantilla

1. Admin selecciona una plantilla.
2. Sistema muestra editor con variables disponibles: `{cliente_nombre}`, `{placa}`, `{monto}`, `{fecha}`.
3. Admin modifica texto y guarda.
4. Sistema almacena versión y aplica a nuevos envíos.

---

## Datos de Entrada — Plantilla

| Campo | Tipo | Descripción | Requerido |
|-------|------|-------------|-----------|
| evento | String | Evento que dispara (entry, exit, payment) | Si |
| canal | Enum | EMAIL, SMS, PUSH | Si |
| titulo | String(100) | Asunto/título | Si |
| cuerpo | Text | Mensaje con variables | Si |
| activa | Boolean | Si está activa | Si |

---

## Criterios de Aceptación

1. Cada evento de notificación puede tener plantilla para email, SMS y push.
2. Las variables se reemplazan dinámicamente en el envío.
3. El admin puede hacer pruebas de envío a su propio email/teléfono.
4. Las plantillas tienen historial de cambios.

---

## Notas

- Los SMS tienen límite de 160 caracteres; el sistema los trunca o convierte a MMS.
- Se debe guardar versión de cada plantilla para auditoría.
